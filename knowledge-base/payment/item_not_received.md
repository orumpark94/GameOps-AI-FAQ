# 결제 후 아이템이 지급되지 않는 경우

## 문의 유형

결제문의

## 사용자 질문 예시

- 결제했는데 아이템이 들어오지 않았어요.
- 구매 완료 문자는 왔는데 게임 안에서 상품이 보이지 않아요.
- 캐시는 차감됐는데 구매한 상품이 지급되지 않았어요.

## 답변 기준

결제 승인 후 아이템 지급까지 최대 10분 정도 지연될 수 있습니다. 게임을 완전히 종료한 뒤
다시 접속하고, 우편함, 보관함, 인벤토리, 상품 수령함을 확인합니다.

결제 내역에서 주문 상태가 `완료`인지 확인합니다. 결제가 `대기`, `취소`, `실패` 상태라면
실제 결제 수단의 승인 또는 취소 내역을 함께 확인합니다.

## 고객센터 문의가 필요한 경우

10분 이후에도 지급되지 않았고 주문 상태가 완료라면 고객센터에 다음 정보를 전달합니다.

- 계정 ID
- 서버명과 캐릭터명
- 상품명과 결제 금액
- 결제 일시
- 결제 영수증 또는 주문번호

영수증에는 주문번호가 보이도록 하되 카드번호 전체, 계좌번호, 비밀번호 등 민감정보는 가립니다.

## 안내 주의사항

챗봇은 실제 결제 내역을 조회할 수 없으며 환불 또는 아이템 지급이 완료되었다고 단정하지 않습니다.
