# 구매한 상품의 환불을 요청하는 경우

## 문의 유형

결제문의

## 사용자 질문 예시

- 잘못 구매한 상품을 환불하고 싶어요.
- 구매한 아이템을 사용하지 않았는데 취소할 수 있나요?
- 환불 신청 방법을 알려주세요.

## 답변 기준

환불 가능 여부는 구매 경로, 상품 사용 여부, 구매 시점, 관련 운영정책에 따라 달라집니다.
게임 내 고객센터 또는 구매한 앱스토어의 주문 내역에서 환불 신청 경로를 확인합니다.

구매한 아이템을 사용하거나 이동하지 말고 주문번호와 구매 상태를 보존합니다. 앱스토어를 통한
결제는 해당 스토어의 환불 절차가 우선 적용될 수 있습니다.

## 고객센터 문의 시 필요한 정보

- 계정 ID, 서버명, 캐릭터명
- 상품명, 구매 일시, 결제 금액
- 주문번호 또는 영수증
- 환불을 요청하는 사유
- 상품 사용 여부

## 안내 주의사항

챗봇은 환불 가능 여부, 환불 금액, 처리 완료 시점을 확정하지 않습니다. 정확한 판단은 결제
기록과 운영정책을 확인한 뒤 이루어집니다.
