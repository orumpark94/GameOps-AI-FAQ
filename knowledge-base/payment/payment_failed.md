# 결제가 실패하거나 계속 진행되지 않는 경우

## 문의 유형

결제문의

## 사용자 질문 예시

- 결제 버튼을 눌러도 실패합니다.
- 결제 처리 중 화면에서 멈췄어요.
- 카드 승인은 됐는데 게임에서는 결제 실패로 나옵니다.

## 답변 기준

네트워크 연결을 확인하고 게임 또는 결제 화면을 완전히 종료한 뒤 다시 시도합니다. 결제 수단의
한도, 잔액, 온라인 결제 차단 여부와 앱스토어 또는 결제 서비스의 로그인 상태를 확인합니다.

실패 직후 반복 결제를 시도하면 중복 승인이 발생할 수 있으므로 먼저 게임 결제 내역과 결제
수단의 승인 내역을 확인합니다.

## 고객센터 문의가 필요한 경우

- 결제 수단에는 승인됐지만 게임 결제 내역이 실패인 경우
- 동일한 오류 코드가 반복되는 경우
- 결제 금액이 차감됐지만 주문번호가 생성되지 않은 경우

문의 시 계정 ID, 상품명, 결제 시각, 결제 수단, 오류 코드, 주문번호 또는 영수증을 전달합니다.

## 안내 주의사항

카드번호 전체, 유효기간, 보안코드, 계좌 비밀번호는 전달하지 않습니다.
