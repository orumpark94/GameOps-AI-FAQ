# 동일한 결제가 중복으로 승인된 경우

## 문의 유형

결제문의

## 사용자 질문 예시

- 같은 상품이 두 번 결제됐어요.
- 결제 오류가 났는데 카드 승인이 두 건 잡혔어요.
- 한 번 구매했는데 영수증이 여러 개입니다.

## 답변 기준

게임의 결제 내역과 결제 수단의 승인 내역에서 주문번호, 결제 시각, 금액을 각각 확인합니다.
일시적인 승인 보류 또는 자동 취소 건은 실제 청구로 확정되지 않을 수 있으므로 결제 상태가
`완료`, `취소`, `대기` 중 무엇인지 확인합니다.

동일한 주문번호가 화면에 반복 표시되는 것인지 서로 다른 주문번호로 실제 결제가 발생한 것인지
구분해야 합니다.

## 고객센터 문의 시 필요한 정보

- 계정 ID, 서버명, 캐릭터명
- 중복으로 보이는 각 주문번호
- 각 결제의 일시와 금액
- 결제 수단 종류
- 민감정보를 가린 영수증 또는 승인 내역

## 안내 주의사항

챗봇은 중복 청구 여부, 취소 완료, 환불 완료를 확정하지 않습니다. 실제 승인 상태는 결제
사업자와 고객센터 확인이 필요합니다.
