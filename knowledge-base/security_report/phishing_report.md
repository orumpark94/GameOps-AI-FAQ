# 게임 운영자를 사칭한 피싱 메시지를 받은 경우

## 문의 유형

해킹/신고

## 사용자 질문 예시

- 운영자라면서 링크를 보냈어요.
- 이벤트 당첨이라며 비밀번호를 요구합니다.
- 게임 아이템을 준다며 외부 사이트 로그인을 유도해요.

## 답변 기준

의심스러운 링크를 열지 말고 비밀번호, 인증번호, 복구 코드, 결제 정보를 입력하지 않습니다.
보낸 이용자를 차단하고 게임 내 신고 기능으로 신고합니다.

이미 링크를 열거나 정보를 입력했다면 안전한 기기에서 게임 계정과 이메일 비밀번호를 변경하고
모든 기기에서 로그아웃합니다. 2차 인증을 설정하고 기기의 악성코드 검사를 실행합니다.

## 고객센터 신고 시 필요한 정보

- 발신자의 서버명과 캐릭터명 또는 계정 정보
- 메시지를 받은 날짜와 시각
- 메시지 전체 내용과 링크 주소
- 원본 스크린샷
- 정보를 입력했는지 여부

## 보안 주의사항

공식 운영자는 채팅이나 외부 사이트를 통해 비밀번호와 인증번호를 요구하지 않습니다.
챗봇은 신고 대상의 제재 여부를 확정하지 않습니다.
