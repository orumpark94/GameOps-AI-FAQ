# 본인이 하지 않은 결제가 확인된 경우

## 문의 유형

해킹/신고

## 사용자 질문 예시

- 제가 구매하지 않은 게임 결제가 있습니다.
- 모르는 주문번호로 결제 문자가 왔어요.
- 계정 도용 후 결제가 발생한 것 같아요.

## 즉시 해야 할 조치

게임 계정과 가입 이메일의 비밀번호를 변경하고 모든 기기에서 로그아웃합니다. 2차 인증을
설정하고 결제 수단의 추가 사용을 차단하기 위해 카드사, 은행 또는 앱스토어에 문의합니다.

게임 결제 내역과 결제 수단의 승인 내역에서 주문번호, 결제 시각, 금액을 확인합니다.

## 고객센터 신고 시 필요한 정보

- 계정 ID, 서버명, 캐릭터명
- 본인이 하지 않은 각 주문번호
- 결제 일시와 금액
- 결제 수단 종류
- 민감정보를 가린 영수증 또는 승인 내역
- 의심 접속 기록이 있다면 해당 날짜와 시각

## 보안 주의사항

카드번호 전체, 계좌번호, 비밀번호, 인증번호는 전달하지 않습니다. 환불과 승인 취소 여부는
결제 사업자 및 고객센터 확인이 필요하며 챗봇이 확정하지 않습니다.
