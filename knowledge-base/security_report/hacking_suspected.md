# 계정 해킹 또는 무단 접속이 의심되는 경우

## 문의 유형

해킹/신고

## 사용자 질문 예시

- 모르는 사람이 제 계정에 접속한 것 같아요.
- 접속하지 않은 시간에 아이템이 사라졌어요.
- 로그인 기록에 모르는 기기가 있습니다.

## 즉시 해야 할 조치

안전한 기기에서 게임 계정과 가입 이메일의 비밀번호를 각각 변경합니다. 같은 비밀번호를 다른
서비스에서도 사용했다면 해당 서비스의 비밀번호도 변경합니다.

모든 기기에서 로그아웃하고 2차 인증을 설정합니다. PC와 모바일 기기의 악성코드 검사를
실행하고 의심스러운 프로그램이나 브라우저 확장 기능을 제거합니다.

## 고객센터 신고 시 필요한 정보

- 계정 ID, 서버명, 캐릭터명
- 의심 접속을 발견한 날짜와 시각
- 마지막으로 정상 접속한 시각
- 사라지거나 변경된 아이템과 계정 정보
- 로그인 알림, 접속 기록, 관련 스크린샷

## 보안 주의사항

비밀번호, 인증번호, 복구 코드는 누구에게도 전달하지 않습니다. 증거를 보존하고 의심되는
접속 기록을 임의로 삭제하지 않습니다.

챗봇은 해킹 여부, 아이템 복구, 가해 계정 제재를 확정하지 않습니다.
