# 욕설, 괴롭힘 또는 부적절한 이용자를 신고하는 경우

## 문의 유형

해킹/신고

## 사용자 질문 예시

- 채팅으로 계속 욕설하는 이용자를 신고하고 싶어요.
- 게임에서 지속적으로 괴롭힘을 당하고 있어요.
- 부적절한 닉네임이나 채팅을 발견했습니다.

## 답변 기준

게임 내 신고 기능에서 신고 대상, 사유, 발생 시각을 정확히 선택합니다. 채팅 차단 또는 이용자
차단 기능을 함께 사용하여 추가 노출을 줄입니다.

원본 맥락을 확인할 수 있도록 신고 대상의 캐릭터명, 서버명, 채팅 내용, 발생 시각이 보이는
스크린샷이나 영상을 보관합니다. 이미지를 임의로 편집하지 않습니다.

## 고객센터 문의 시 필요한 정보

- 신고 대상의 서버명과 캐릭터명
- 발생 날짜와 시각
- 발생한 채널, 지역 또는 콘텐츠
- 신고 사유와 상황 설명
- 스크린샷 또는 영상

## 안내 주의사항

제재 여부와 수위는 운영정책과 내부 확인 결과에 따라 결정됩니다. 챗봇은 특정 이용자의 제재
또는 제재 기간을 확정하지 않습니다.
