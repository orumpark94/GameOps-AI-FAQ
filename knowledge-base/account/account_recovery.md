# 계정 정보를 잃어버려 계정을 찾을 수 없는 경우

## 문의 유형

계정문의

## 사용자 질문 예시

- 어떤 이메일로 가입했는지 모르겠어요.
- 휴대전화를 바꿔서 기존 계정에 로그인할 수 없어요.
- 오래전에 사용한 계정을 복구하고 싶어요.

## 답변 기준

로그인 화면의 계정 찾기 기능에서 본인인증을 진행하여 가입 계정을 확인합니다.
연동 로그인 계정이라면 사용했던 플랫폼 계정의 로그인 상태와 연동 여부를 확인합니다.

계정 찾기 기능으로 확인되지 않거나 기존 인증 수단을 사용할 수 없다면 고객센터의 본인확인
절차가 필요합니다.

## 고객센터 문의 시 필요한 정보

- 기억하는 계정 ID 또는 가입 이메일 일부
- 서버명과 캐릭터명
- 대략적인 가입 시기와 마지막 접속 시기
- 본인이 생성한 캐릭터 또는 이용 기록
- 본인 소유임을 확인할 수 있는 결제 주문번호가 있는 경우 해당 번호

고객센터가 요구하지 않은 주민등록번호, 신분증 전체 이미지, 비밀번호는 먼저 전달하지 않습니다.

## 안내 주의사항

제공한 정보만으로 계정 소유권이 확인되지 않을 수 있습니다. 챗봇은 계정 복구 가능 여부나
복구 완료를 확정하지 않습니다.
