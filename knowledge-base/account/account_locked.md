# 로그인 실패로 계정이 잠긴 경우

## 문의 유형

계정문의

## 사용자 질문 예시

- 비밀번호를 여러 번 틀려서 계정이 잠겼어요.
- 로그인 시도 횟수를 초과했다는 메시지가 나옵니다.
- 계정 잠금을 바로 해제할 수 있나요?

## 답변 기준

반복된 로그인 실패가 감지되면 계정 보호를 위해 일시적으로 로그인이 제한될 수 있습니다.
로그인 시도는 중단하고 안내된 잠금 시간이 지난 뒤 다시 시도합니다.

비밀번호가 기억나지 않으면 잠금이 해제된 뒤 비밀번호 찾기 기능을 이용합니다. 다른 기기나
네트워크에서도 동일한 문제가 발생하는지 확인하되 짧은 시간에 반복해서 로그인하지 않습니다.

## 고객센터 문의가 필요한 경우

- 안내된 시간이 지나도 잠금이 해제되지 않는 경우
- 본인이 로그인하지 않았는데 실패 알림이 반복된 경우
- 계정 이메일 또는 휴대전화가 변경된 경우

문의 시 계정 ID, 서버명, 캐릭터명, 오류 메시지, 마지막 정상 로그인 시각을 전달합니다.

## 보안 주의사항

본인이 시도하지 않은 로그인 기록이 의심되면 비밀번호를 변경하고 2차 인증을 설정합니다.
챗봇은 계정 잠금이 해제되었다고 단정하지 않습니다.
