# 가입 이메일 또는 휴대전화 정보를 변경하는 경우

## 문의 유형

계정문의

## 사용자 질문 예시

- 가입 이메일을 변경하고 싶어요.
- 휴대전화 번호가 바뀌었어요.
- 기존 이메일을 사용할 수 없는데 변경할 수 있나요?

## 답변 기준

계정 설정의 개인정보 또는 보안 메뉴에서 현재 인증 수단으로 본인인증을 완료한 뒤 이메일이나
휴대전화 번호를 변경합니다.

변경 확인 메일 또는 인증번호가 도착하지 않으면 스팸함, 수신 차단 설정, 입력한 주소와 번호를
확인합니다.

## 고객센터 문의가 필요한 경우

- 기존 이메일과 휴대전화에 모두 접근할 수 없는 경우
- 본인이 요청하지 않은 정보 변경 알림을 받은 경우
- 변경 기능에서 오류가 반복되는 경우

문의 시 계정 ID, 서버명, 캐릭터명, 변경하려는 정보의 종류와 오류 화면을 전달합니다.

## 보안 주의사항

이메일 비밀번호, 문자 인증번호, 게임 계정 비밀번호는 전달하지 않습니다.
보안을 위해 계정 소유권 확인 전에는 정보 변경이 제한될 수 있습니다.
