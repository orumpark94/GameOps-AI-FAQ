# 비밀번호를 잊어버린 경우

## 문의 유형

계정문의

## 사용자 질문 예시

- 비밀번호를 잊어버렸어요.
- 로그인이 되지 않는데 비밀번호를 어떻게 바꾸나요?
- 비밀번호 재설정 메일이 오지 않아요.

## 답변 기준

로그인 화면의 `비밀번호 찾기` 기능에서 가입한 이메일 또는 휴대전화 본인인증을 진행한 뒤
새 비밀번호를 설정합니다.

재설정 메일이 도착하지 않으면 스팸 메일함, 메일 수신 차단 설정, 가입 시 입력한 이메일 주소를
확인합니다. 인증번호가 도착하지 않으면 통신 상태와 문자 수신 차단 설정을 확인한 뒤 잠시 후
다시 요청합니다.

## 고객센터 문의가 필요한 경우

- 가입한 이메일 또는 휴대전화를 더 이상 사용할 수 없는 경우
- 본인인증을 완료할 수 없는 경우
- 여러 번 시도한 뒤 계정이 잠긴 경우
- 본인이 요청하지 않은 비밀번호 변경 알림을 받은 경우

고객센터 문의 시 계정 ID, 캐릭터명, 서버명, 가입 시기 등 계정 식별에 필요한 정보를 안내합니다.

## 보안 주의사항

현재 비밀번호, 새 비밀번호, 인증번호는 챗봇이나 상담원에게 전달하지 않습니다.
챗봇은 비밀번호 변경 또는 계정 복구가 완료되었다고 단정하지 않습니다.
