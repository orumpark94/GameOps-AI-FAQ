# 게임 접속이 끊기거나 서버에 연결할 수 없는 경우

## 문의 유형

게임문의

## 사용자 질문 예시

- 서버에 접속할 수 없어요.
- 게임 중 계속 연결이 끊깁니다.
- 서버 선택 후 로딩 화면에서 멈춰요.

## 답변 기준

공식 공지에서 점검 또는 장애 여부를 먼저 확인합니다. 게임을 종료하고 네트워크 장비와 기기를
재시작한 뒤 다시 접속합니다.

유선 연결 또는 안정적인 네트워크를 사용하고 VPN, 프록시, 다운로드 프로그램을 종료합니다.
게임 클라이언트의 파일 검사 기능이 있다면 손상된 파일을 확인합니다.

## 고객센터 문의 시 필요한 정보

- 이용 중인 서버와 캐릭터명
- 접속을 시도한 날짜와 시각
- 표시된 오류 코드와 오류 메시지
- 사용 기기, 운영체제, 네트워크 종류
- 문제가 발생하는 화면의 스크린샷

## 안내 주의사항

특정 계정에서만 발생하는지, 같은 네트워크의 다른 기기에서도 발생하는지를 함께 확인하면
원인 분석에 도움이 됩니다.
