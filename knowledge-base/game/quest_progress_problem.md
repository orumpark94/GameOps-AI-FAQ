# 퀘스트가 진행되지 않거나 보상을 받을 수 없는 경우

## 문의 유형

게임문의

## 사용자 질문 예시

- 퀘스트 조건을 완료했는데 진행되지 않아요.
- NPC에게 말을 걸어도 보상을 받을 수 없어요.
- 퀘스트 아이템이 나오지 않습니다.

## 답변 기준

퀘스트 설명의 선행 조건, 요구 레벨, 대상 지역, 파티 상태, 제한 횟수를 다시 확인합니다.
퀘스트 추적을 해제했다가 다시 설정하고 지역 이동 또는 재접속 후 진행 상태를 확인합니다.

인벤토리 공간이 부족하거나 보상 수령 조건을 충족하지 못한 경우 보상이 지급되지 않을 수
있습니다. 공식 공지의 알려진 문제 목록도 확인합니다.

## 고객센터 문의 시 필요한 정보

- 서버명과 캐릭터명
- 퀘스트 이름과 현재 단계
- 문제가 발생한 날짜와 시각
- 완료한 조건과 표시된 메시지
- 진행 화면의 스크린샷 또는 영상

## 안내 주의사항

챗봇은 퀘스트 완료 처리나 보상 지급을 직접 수행할 수 없습니다.
