# 캐릭터를 실수로 삭제한 경우

## 문의 유형

게임문의

## 사용자 질문 예시

- 캐릭터를 실수로 삭제했어요.
- 삭제한 캐릭터를 복구할 수 있나요?
- 캐릭터가 목록에서 사라졌습니다.

## 답변 기준

먼저 로그인한 서버와 계정이 맞는지 확인하고 캐릭터 목록의 삭제 대기 또는 복구 메뉴를
확인합니다. 삭제 유예기간이 제공되는 경우 해당 기간 안에 직접 복구할 수 있습니다.

직접 복구 기능이 없거나 유예기간이 지난 경우 고객센터 확인이 필요합니다.

## 고객센터 문의 시 필요한 정보

- 계정 ID와 서버명
- 삭제한 캐릭터명과 직업
- 삭제한 날짜와 대략적인 시각
- 마지막 정상 접속 시각
- 삭제 원인

## 안내 주의사항

삭제 후 경과 시간, 캐릭터 데이터 보존 상태, 운영정책에 따라 복구가 불가능할 수 있습니다.
챗봇은 복구 가능 여부 또는 복구 완료를 확정하지 않습니다.
